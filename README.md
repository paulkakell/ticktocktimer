 
This project is a countdown timer multi-user docker application that can use either an internal or external database. It provides a simple and easy-to-use countdown timer functionality that can be accessed by multiple users. It includes api services as well as connection to notification services via webhook or api.

### Main Function Points
- Countdown timer functionality
- Multi-user support
- Option to use either an internal or external database

### Technology Stack
- Python
- HTML
- CSS
- Docker
